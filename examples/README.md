# セクションテンプレート

ランディングページ作成時に参考にできるセクション別テンプレート集です。

## 構成

### hero-section/
- **用途**: ページトップのヒーロー領域
- **参考実装**:
  - `app/accounting/page.tsx` (行77-118)
  - `app/hr/page.tsx` (行65-124)
  - `app/launch/page.tsx` (行75-116)
- **カスタマイズポイント**:
  - グラデーション色
  - 左側テキスト（タイトル、説明、CTA）
  - 右側の統計情報やダッシュボード

### features-section/
- **用途**: 機能や特徴を一覧表示
- **参考実装**:
  - `app/accounting/page.tsx` (行151-201): 番号付き交互レイアウト
  - `app/hr/page.tsx` (行159-189): 3列グリッド
- **提供パターン**:
  - `FeaturesGridSimple()`: 3列グリッド表示
  - `FeaturesAlternatingLayout()`: 番号付き交互レイアウト
- **カスタマイズポイント**:
  - アイコン
  - グリッド列数
  - 背景グラデーション

### steps-section/
- **用途**: プロセスや手順を段階的に表示
- **参考実装**: `app/launch/page.tsx` (行118-151)
- **カスタマイズポイント**:
  - ステップ数
  - 各ステップの所要時間
  - コネクタ線の色

### comparison-section/
- **用途**: 自社サービスと競合・従来方法を比較
- **参考実装**: `app/launch/page.tsx` (行238-279)
- **カスタマイズポイント**:
  - 比較項目
  - テーブルヘッダーの色
  - 列数（3列以上に拡張可能）

### faq-section/
- **用途**: よくある質問をアコーディオン形式で表示
- **参考実装**: `app/accounting/page.tsx` (行281-308)
- **カスタマイズポイント**:
  - Q&A の内容
  - 背景色
  - 矢印アイコン

### pain-points-section/
- **用途**: 問題と解決を Before/After で対比表示
- **参考実装**: `app/hr/page.tsx` (行126-157)
- **カスタマイズポイント**:
  - Before/After のテキスト
  - テキスト装飾（color, line-through など）
  - グリッド列数

### merits-section/
- **用途**: サービスの主要メリットをハイライト付きで表示
- **参考実装**: `app/launch/page.tsx` (行153-188)
- **カスタマイズポイント**:
  - メリット内容
  - ハイライト値（数字やテキスト）
  - アイコン

### testimonial-section/
- **用途**: ユーザーやクライアントからの推薦文を表示
- **参考実装**: `app/hr/page.tsx` (行248-275)
- **提供パターン**:
  - `TestimonialSectionSimple()`: 単一の推薦文
  - `TestimonialSectionGrid()`: 複数の推薦文をグリッド表示
- **カスタマイズポイント**:
  - 推薦文の内容
  - ユーザー情報（名前、肩書き）
  - アバター色

## 使用方法

1. 新しいページを作成する際は、このディレクトリから該当するセクションテンプレートを参考に
2. 各ファイルのコメントを確認して、カスタマイズポイントを把握
3. 参考実装を確認して、実装パターンを理解
4. 必要に応じてコピーして、ページに組み込む

## 注記

- これらはテンプレートであり、実装を強制するものではありません
- ページごとのニーズに合わせて自由にカスタマイズしてください
- 新しいセクションパターンが必要になった場合は、このディレクトリに追加してください
